## 💻 开发

### 基本

- Typescript
- Vue 3
    - Vite
    - Pinia
    - Vue Router
    - Axios
- SCSS
- Element Plus

### 代码规范化工具

- ESLint
- Prettier

### 其它开发依赖

- lodash.clonedeep
- spacejs

## 🦾 源码

### 1. 安装依赖

```npm
pnpm install
```

### 2. 启动开发环境

```npm
pnpm run dev
```

### 3. 开发

```shell
# 提交代码

# 1.添加暂存区
git add .

# 2.提交代码
  # `type`的允许值详见 commitlint.config.js
git commit -m <type>[optional scope]: <description>

# 3.推送
git push
```
